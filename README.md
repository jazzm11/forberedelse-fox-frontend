# 🦊 Fox Voting Frontend

This is the frontend part of the Fox Voting App using **EJS templates**.

---

## 🚀 Pages

### 🏠 Home (Voting Page)
- Shows two fox images
- User votes by clicking a button on one fox

Route:

GET /
    /stats

POST /vote


---

### 📊 Stats Page
- Shows foxes sorted by votes
- Displays most popular foxes

Route:

GET /stats


---

## 🎨 UI Features

- Dark modern theme
- Orange accent color
- Simple responsive layout
- Minimal design
